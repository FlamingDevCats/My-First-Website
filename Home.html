Alright, so you wanna go play games on the school computers by using your pc/laptop at home? Well if you are going to, you are definitely going to need some basic things at least which I will list below later.

Skip this paragraph if you hate reading

Using Chrome remote desktop is something you would need to install on yor computer. But to use it effectively you would need good internet and some other things. There are many things you would be able to do with Chrome remote desktop, like playing games, checking something running on your computer etc. You could play anything installed on your computer. You would also be able to get files on your computer to the device you would be using with chrome remote desktop. Also, it is impossible for the school () to know what games, websites or whatever you are doing unless they can see your screen, which does actually happen due the program they use to take control of your screen.

If you want to know how to revert things you did (like reverting the installation Chrome Remote Desktop), I will include tutorials for that. Also this whole tutorial is aimed for windows users as I don't have a way to do this with macos plus I don't have the time to make a macos version. I this tutorial I am still using Windows 10 on a laptop but it should still work on Windows 11.

So, here are the requirements listed:

  • Good internet at home
  • No one using your computer while you are at school*
  • Admin access on your computer
  • Can keep your computer on for the duration of school
  • Able to keep it connected to internet for the duration of school
  • Have enough time to finish and troubleshoot potential issues you might have
*Who would want to use a computer and see that it is doing stuff on it's "own"?
