Installing a Virtual Monitor

Alright so this is where we will be installing a virtual monitor. This can make it seem impossible to see your computer screen (because it is on the virtual monitor) but I will include help for fixing stuff like that in another tutorial.

This tutorial is completely optional if you don't want to play games using Chrome Remote Desktop. This is for the case you turn off your pc screen or close the lid. If you keep your monitor on or you keep your laptop lid open, you can skip this. The reason why we need a virtual monitor is because you need a way for your gpu to actually render things while it is inactive (which happens when your monitor is off or your laptop lid is closed). If you are wondering why some things still render without a virtual monitor, it is because those are done by software rendering (rendering on the cpu).

  1. Go to the virtual monitor releases page.         (Source code)
  2. Download the zip that doesn't say Driver Only and don't download the source code. In this example, the correct zip is at the top of the Assets section. I recommend that you download one that is in the latest release. These are marked with the green Latest tag and are usually at the top.
  3. Extract the zip to a folder. You can put this folder anywhere.
  4. Open the folder and run the executable. If you get problems like vcruntime140.dll not found, follow these steps.
    1. Go to Microsoft Visual C++ Redistributable page.
    2. Click on the latest version
    3. Download and run the x86 and x64 versions. When you run the files, follow the on-screen instructions. Also, if you can't install/run the x86 version, just leave it.
